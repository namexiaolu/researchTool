# 市面上的一维多相流模拟项目收集

## 总览：一维多相流项目的分类

一维多相流模拟可分为三大类：

- **油藏模拟（Reservoir Simulation）**：1D/2D/3D 多孔介质中流体行为
- **井筒/管道模拟（Wellbore/Pipeline Flow）**：1D 管流中的多相流动
- **通用计算库**：物性计算、数值框架

---

## 二、井筒/管道 1D 流动模拟

### 2.1 综合/生产级

| 项目                                              | 语言         | 许可证        | 特点                                                                        |
| ----------------------------------------------- | ---------- | ---------- | ------------------------------------------------------------------------- |
| [Marlim3](https://github.com/petrobras/marlim3) | Python/C++ | Apache-2.0 | **Petrobras 官方出品**，1D 瞬态/稳态多相流，管网+气举+人工举升+组分流体库，性能高（C++/Fortran 内核），中学习曲线 |
| [DWell](https://gitlab.com/open-darts/dwell)    | Python     | Apache-2.0 | 多相多组分非等温管道流，可与 [[DARTS]] 油藏模拟器耦合                                          |
| [T2Well](https://tough.lbl.gov)                 | C++        | BSD        | 1D 井筒+3D 油藏耦合，瞬态多相多组分，CO₂ 注入适用                                            |
| [pyfas](https://github.com/gpagliuca/pyfas)     | Python     | GPL-3.0    | **OLGA 风格**瞬态多相流管道，44+ stars，可作为 OLGA 输出的可视化插件                            |

## 七、关键对比

### 核心项目

| 维度 | OPM Flow | MRST | Marlim3 | PyReSim |
|------|----------|-----|--------|---------|
| 维度 | 1D/2D/3D | 1D/2D/3D | 1D 为主 | 1D/2D |
| 场景 | 油藏全流程 | 研究原型 | 管道/管网 | 教学/轻量 |
| 性能 | 工业级（C++） | 中等（MATLAB） | 高（C++/Fortran） | 低（Python） |
| 学习曲线 | 陡 | 中 | 中 | 低 |
| 社区活跃度 | 高（OPM 组织） | 高（SINTEF） | 中（Petrobras） | 低 |

## 八、推荐方案速查

| 场景          | 推荐方案                              | 理由                      |
| ----------- | --------------------------------- | ----------------------- |
| 工业级油藏模拟     | **OPM Flow**                      | Eclipse 兼容、验证充分、社区活跃    |
| 学术研究/快速原型   | **MRST**                          | 灵活度高、文档丰富、可定制           |
| 管道/管网工程     | **Marlim3**                       | Petrobras 出品、瞬态+稳态、开箱即用 |
| Python 教学入门 | **PyReSim / Reservoir-Simulator** | 纯 Python、易于理解           |
| OLGA 替代/后处理 | **pyfas**                         | OLGA 风格、可做可视化插件         |
