# 概念和解释汇总（按主题分组）

## 流体基础

| 概念名称                    | 基本含义           | 详细含义                                                 | 适用场景                     |
| ----------------------- | -------------- | ---------------------------------------------------- | ------------------------ |
| 稳态                      | 状态不随时间变化       | 只看某一固定工况下的压力、温度、流量等空间分布，时间导数近似为 0                    | 管径选型、方案设计、长期平均压降/温降、产能优化 |
| 瞬态                      | 状态随时间变化        | 需要按时间步模拟启停、阀门动作、段塞、水击、停输再启动等动态过程                     | 安全校核、流动保障、在线监控、风险预警      |
| 持液率                     | 截面中液体占比        | 表示管道内液体占据的体积分数                                       | 段塞分析、积液判断、流型识别           |
| 流型                      | 流动形态分类         | 如分层流、泡流、段塞流、环状流等                                     | 流动保障、可视化、风险分析            |
| 段塞流                     | 液体周期性聚集并冲击下游   | 常见于多相管道，会引发压力波动和设备冲击                                 | 立管、海底管道、分离器负荷校核          |
| 水击                      | 压力冲击波现象        | 阀门快速关断或泵跳机时产生瞬时高压                                    | 管道承压校核、设备保护              |
| 水合物                     | 低温高压下形成的晶体堵塞物  | 气体与水在特定条件下形成固体，易堵管                                   | 深水、低温管线、停输再启动            |
| 蜡沉积                     | 原油低温析蜡并附着管壁    | 会增加阻力、降低通径并引发堵塞                                      | 高蜡原油管线、停输重启              |
| 流动保障                    | 保证管道可安全稳定输送    | 覆盖段塞、水合物、蜡沉积、积液、冻堵等问题                                | 油气长输、海底管道、生产系统           |
| 清管                      | 管内清除积液和杂质      | 通过清管球/清管器推动管内液体和沉积物                                  | 管道维护、流动保障、停输恢复           |
| 气举                      | 注入气体辅助举升液体     | 通过向井筒或管线注入高压气体，降低混合流体密度、改善流动状态，从而帮助原油或液体更容易流动和提升     | 低产井举升、井筒生产优化、油田生产系统设计    |
| VGL（气举阀，Gas Lift Valve） | 控制气举注入的阀门      | 安装在气举系统或井筒相关位置，通过压力差或设定条件自动开启/关闭，控制注入气体进入井筒或环空的流量与时机 | 气举井、注气控制、井筒举升、环空服务线      |
| 注气                      | 向系统中注入气体       | 向井筒、油藏、管线或工艺系统中注入天然气、氮气、二氧化碳等气体，以实现举升、增压、驱替、保压或测试等目的 | 气举生产、油藏驱替、管线吹扫、工艺调试      |
| 循环                      | 让流体按回路持续流动     | 通过井筒、管线或设备形成进出回路，使流体连续循环，用于清洗、换液、带出杂质、传热或维持工况稳定      | 井筒作业、洗井、试压、工艺回路运行        |
| 环空服务线                   | 为环空提供流体通道的辅助管线 | 指连接井筒环空或相关设备、用于注气、注液、放空、测试或监测的专用服务管线，常承担辅助操作而非主生产通道  | 气举井、井口流程、井筒作业、监测与维护      |
| 环空                      | 套管与油管之间的空间     | 井筒内油管外侧、套管内侧形成的环形空间，可用于注气、注液、测试或压力监测                 | 气举、完井、井筒测试、环空压力管理        |
| 油管                      | 井筒内输送产液的管柱     | 井下用于把油、气、水从井底输送到井口的管子，也叫生产管柱                         | 采油、举升、井筒生产               |
| 套管                      | 用于支撑和隔离井筒的外管   | 井筒中更外层的钢管，用于支撑井壁、隔离地层、保护井眼稳定                         | 井身结构、完井、地层隔离             |
| 井筒                      | 井从井口到井底的通道     | 油气井内部的垂向或近垂向流动通道，连接地层、管柱和井口设备                        | 采油采气、举升、测试、作业            |
| 井口                      | 井的地面连接和控制部位    | 井筒在地面的接口，通常包含阀门、压力表、节流和安全控制设备                        | 生产控制、注采切换、井口作业           |
| 分离器                     | 分离油气水的工艺设备     | 利用密度差、停留时间和结构设计，把混合流中的气、油、水分开                        | 站场处理、气液分离、液位控制           |
| 节流阀                     | 用于调节流量和压降的阀门   | 通过改变开度控制流体流量、压力和流动状态                                 | 井口节流、管网调压、工况控制           |
| 注醇                      | 向系统注入甲醇等抑制剂    | 向管线或井筒加入甲醇等化学品，用于防冻、防水合物或清除堵塞风险                      | 水合物抑制、低温管线、停输重启          |
| 放空                      | 将系统内气体安全排出     | 通过放空阀或放空管线把设备、管线或环空内的气体释放到安全位置                       | 维压检修、紧急泄放、测试作业           |
| 热损失                     | 管内流体向环境散热      | 流体与土壤、空气、海水之间发生传热                                    | 温降预测、保温设计、冻堵风险分析         |
| 高程剖面                    | 管道沿线的海拔变化      | 决定重力项、积液低点和段塞易发位置                                    | 地面管网、海底管线、流动保障           |
| 粗糙度                     | 管壁表面粗糙程度       | 影响摩擦因子、压降和流动阻力                                       | 管道设计、老化评估、压降计算           |
| 局部损失系数 K                | 构件引起的额外压降参数    | 用于描述弯头、阀门、三通等局部阻力                                    | 1D 模型、管件校准               |
| PVT                     | 压力-体积-温度物性     | 描述油气水组分在不同工况下的热力学性质                                  | 多相流建模、相态计算               |
| EOS                     | 状态方程           | 用于计算气液平衡、密度、压缩因子等                                    | 组分流体模拟、相态分析              |

## 数值模型

| 概念名称    | 基本含义         | 详细含义                            | 适用场景                               |
| ------- | ------------ | ------------------------------- | ---------------------------------- |
| 一维多相流   | 沿管道轴向建模      | 用截面平均变量描述管网，不解析完整三维细节，适合长距离系统计算 | 油气集输、长输管网、实时数字孪生                   |
| 三维 CFD  | 三维流场精细模拟     | 解析速度场、压力场、相界面、涡结构等局部细节          | 弯头、三通、阀门、立管、局部机理分析                 |
| 有限体积法   | 守恒方程离散方法     | 将空间划分为控制体，通过通量平衡求解流体方程          | OpenFOAM、Fluent、STAR-CCM+ 等 CFD 工具 |
| SPH 粒子法 | 基于粒子的流体模拟    | 用粒子表示流体，通过邻域相互作用更新位置和速度         | 实时视觉演示、局部流体效果、交互展示                 |
| 均相模型    | 把多相流等效为单一混合物 | 用等效密度和粘度近似油气水混合流动               | MVP 原型、简化计算、快速演示                   |
| 漂移通量模型  | 描述多相滑移的简化模型  | 在均相模型基础上增加气液或相间相对速度描述           | 简化多相流、持液率估算、流型判断                   |
| 双流体模型   | 各相分别守恒求解     | 对气相、油相、水相分别建立质量、动量、能量方程         | 工程级多相流、OLGA/LedaFlow 类模型           |
| 质量守恒    | 物质不凭空产生或消失   | 某段管道内质量变化等于流入流出差加源项             | 1D 求解、3D CFD、网络平衡                  |
| 动量守恒    | 力与流动变化的平衡    | 压力梯度、重力、摩擦、局部损失共同决定流速和压降        | 压降计算、瞬态波动、设备影响分析                   |
| 能量守恒    | 热量和内能变化守恒    | 考虑流动输运、散热、相变潜热和摩擦生热             | 温降、水合物风险、热力仿真                      |
| 摩擦因子    | 管壁摩擦阻力参数     | 决定沿程压降大小，受雷诺数和粗糙度影响             | 压降计算、管径校核、模型校准                     |
| 沿程压降    | 流体沿管道产生的连续压降 | 由摩擦、重力和流体特性共同决定                 | 管网计算、泵选型、运行评估                      |
| 节点质量平衡  | 节点进出流量守恒     | 任一节点的流入、流出和源汇应满足平衡              | 管网求解、分支分配、配产分析                     |
| 边界条件    | 求解器外部输入约束    | 指入口流量、入口压力、出口背压、温度、阀位等工况输入      | 稳态计算、瞬态计算、实时仿真                     |
| 初始条件    | 瞬态计算起点       | 给定仿真开始时的压力、温度、流量、持液率等初始状态       | 瞬态启动、停输重启、在线滚动预测                   |
| 稳态初始化   | 先求一个稳定初始场    | 瞬态仿真前常先用稳态求解得到合理起始分布            | 瞬态求解、在线系统初始化                       |
| 准瞬态     | 介于稳态和完整瞬态之间  | 考虑一定时间变化，但模型复杂度低于完整瞬态           | 演示级在线预测、轻量状态变化模拟                   |

## 仿真工具

| 概念名称 | 基本含义 | 详细含义 | 适用场景 |
|---|---|---|---|
| OpenFOAM | 开源 CFD 工具箱 | 适合三维或局部高保真离线仿真，可自定义求解器 | 局部流场分析、可视化素材生成、模型标定 |
| OLGA | 商用一维瞬态多相流求解器 | 油气行业常用的动态管网模型，擅长段塞、清管、启停等瞬态 | 流动保障、在线数字孪生、生产运行分析 |
| PIPESIM | 商用一维稳态多相流求解器 | 主要做稳态设计和校核，常与 OLGA 配合使用 | 前期设计、稳态优化、产能分析 |
| LedaFlow | 高精度一维瞬态多相流求解器 | 强调更精细的多相流物理模型，适合复杂场景 | 深水、CO2、氢气混输、高精度瞬态分析 |
| RTTM | 实时瞬态模型 | 结合实时数据滚动计算管道状态和风险 | 在线监控、泄漏检测、预测预警 |

## 数字孪生架构

| 概念名称 | 基本含义 | 详细含义 | 适用场景 |
|---|---|---|---|
| 数字孪生 | 现实系统的虚拟映射 | 结合模型、数据和可视化，对实体系统进行同步描述 | 在线监控、预测、回放、决策支持 |
| 1D 管网模型 | 节点+管段的系统级模型 | 把管网抽象成计算网络，输出全局压力、流量、温度、持液率等 | 油田地面管网、在线主干、状态机 |
| 1D 在线主干 | 实时系统级计算层 | 负责全管网状态和未来预测，前端只做显示 | 地面管网数字孪生架构 |
| OpenFOAM 离线库 | 局部高保真参考库 | 预计算典型构件和工况，用于标定和缓存 | 局部细节增强、Unity 可视化 |
| Unity 显示层 | 三维交互前端 | 负责颜色、粒子、告警、趋势图和操作界面 | 演示系统、操作台、可视化 |
| 管网拓扑 | 管网的连接关系 | 用节点、管段、设备描述系统的结构和流动路径 | 管网建模、状态映射、求解器输入 |
| PipeState | 管段运行状态数据 | 表示某条管段在某时刻的压力、温度、流量、流速、持液率、流型和风险等级 | Unity 显示、后端通信、历史回放 |
| PipeNode | 管网节点对象 | 表示井口、汇管、阀门连接点、站场边界等连接节点 | 拓扑建模、边界条件、网络计算 |
| PipeEdge | 管网管段对象 | 表示连接两个节点的一段管道，包含长度、直径、粗糙度等属性 | 1D 求解、状态输出、前端映射 |
| PipeDevice | 管网设备对象 | 表示阀门、泵、压缩机、分离器等设备及其特性 | 设备建模、操作模拟、控制联动 |
| 数据映射层 | 后端状态到前端表现的转换层 | 把压力、温度、流量等转换成颜色、粒子、液位、箭头和告警 | Unity 实时显示、可视化系统 |
| 状态插值 | 在两次状态之间平滑过渡 | 让低频后端数据在高帧率前端中连续显示 | 实时可视化、动画平滑 |
| 缓存查表 | 用预计算结果替代实时重算 | 把离线计算结果按工况编号存储，运行时直接读取或插值 | OpenFOAM 数据复用、实时显示 |
| 关键帧缓存 | 离线保存若干时刻结果 | 运行时通过插值生成连续动画，减少实时计算负担 | 局部流场播放、三维演示 |
| 历史回放 | 按历史数据重现运行过程 | 用历史状态、报警和趋势回放过去工况 | 事故复盘、培训、演示 |
| 虚拟计量 | 用模型估算不可直接测量量 | 结合实时数据和模型推断流量、持液率等变量 | 数字孪生、在线监测、成本优化 |
| Look Ahead | 前瞻预测 | 基于当前状态向未来一段时间滚动预测系统变化 | 在线预警、操作决策、调度支持 |
| What-If | 假设工况预演 | 在不下发真实控制前先模拟某个操作可能造成的后果 | 操作培训、风险评估、控制确认 |

## 数据与通信

| 概念名称 | 基本含义 | 详细含义 | 适用场景 |
|---|---|---|---|
| SCADA | 数据采集与监控系统 | 提供压力、温度、流量、阀位等实时边界数据 | 数字孪生、运行监测、控制联动 |
| WebSocket | 长连接数据推送协议 | 适合实时持续发送管段状态和告警信息 | Unity 与后端实时通信 |
| gRPC | 高性能接口协议 | 适合结构化、高频率、强类型的服务通信 | 后端服务、状态订阅、控制接口 |
| OPC UA | 工业互联协议 | 支持读写、订阅、认证、审计，常用于工业系统间通信 | SCADA、DCS、控制服务、数字孪生接入 |
| Historian | 工业历史时序库 | 存储长期运行数据、趋势和事件记录 | 历史回放、报表、追溯分析 |
| 时序数据库 | 按时间组织的数据存储 | 用于保存高频传感器数据和仿真结果 | 监控、趋势分析、回放 |
| 关系数据库 | 结构化业务数据存储 | 保存管网拓扑、设备台账、参数配置和操作记录 | 配置管理、模型管理、业务系统 |
| GIS 数据库 | 地理信息数据库 | 保存空间位置、线路走向、地理属性等信息 | 长输管网、地面设施地图、资产管理 |

## 工业控制

| 概念名称          | 基本含义         | 详细含义                         | 适用场景                |
| ------------- | ------------ | ---------------------------- | ------------------- |
| 控制服务          | 操作命令的中间层     | 负责权限、校验、审计、限幅、排队和下发，不让前端直控现场 | 工业控制、数字孪生操作闭环       |
| CommandIntent | 控制意图对象       | 表达操作者想执行的动作，而非立即直接下发控制       | 预演、审批、二次确认、操作日志     |
| 权限校验          | 验证操作者是否有操作资格 | 按角色、设备和操作范围限制可执行动作           | 生产系统、控制系统、审计        |
| 二次确认          | 关键操作的再次确认机制  | 在高风险控制前要求人工复核和确认             | 阀门操作、停输、启泵、联锁相关操作   |
| 审计日志          | 可追踪的操作记录     | 记录谁在何时做了什么操作以及结果如何           | 工业合规、事故追踪、责任界定      |
| 联锁            | 安全保护逻辑       | 当工况超限时自动阻止危险动作或触发保护          | DCS、SIS、设备保护、安全控制   |
| SIS           | 安全仪表系统       | 独立于常规控制系统的安全保护系统             | 紧急停车、超压保护、高风险工况     |
| DCS           | 分布式控制系统      | 工业现场的主要过程控制系统                | 站场控制、设备联动、监控        |
| PLC           | 可编程逻辑控制器     | 执行现场设备级逻辑控制和联锁               | 阀门、泵、传感器、设备控制       |
| HMI           | 人机界面         | 操作员与工业系统交互的可视化界面             | Unity 操作台、站控界面、监控系统 |
